# Paragon SKU Matching Prototype

Open [index.html](/Users/kathu/Desktop/Paragon%20Deliverables/index.html) in a browser to use the demo.

## What it is

This is a static prototype for a one-shot industrial SKU matching workflow. It simulates how an operator-facing system could:

- parse messy incoming requests
- retrieve a narrowed candidate set
- rank matches using structured and contextual signals
- decide whether to auto-match or escalate to review

## What to try

- `Returning brass fitting request`
- `Ambiguous valve vs fitting`
- `DIN fastener request`
- `New customer, sparse ask`

## Notes

- The catalog and customer data are synthetic.
- The scoring logic is intentionally explainable and visible in the UI.
- The prototype is optimized for clarity of thinking rather than production scale.
